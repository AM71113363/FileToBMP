build.bat will work only with Dev-C++ 4.9.9.*


run FileToBMP.exe (32Bit Windows) 

1.Drag-Drop a file (example a Zip file,you can protect the Zip with a password)
2.DONE

# the app will create a BMP file with the same name in the DIR where the DropFIle is.
example:
         "myfile.zip"  >>    "myfile.bmp"
         "myfile.pdf"  >>    "myfile.bmp"
         "myfile"      >>    "myfile.bmp"


How It Works:

USER( __A__ )
              1. Convert a file to BMP.
              2. Share the IMAGE using Social-Media.
              

USER( __B__ )
              1. Download the IMAGE to PC/phone.
              2. Use a BinaryEditor and remove the first(54 Bytes).
              3. USER( __B__ ) must know the EXT of the original FILE.
              4. Change the EXT.
              

NOTE:

if the Social-Media >> resize the IMAGE file >> or >> Convert it to another format,
USE another Social_Media.
